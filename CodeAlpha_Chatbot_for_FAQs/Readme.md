# ✨ AURON – Generative AI & LLMs Oracle

AURON is a desktop-based AI chatbot that answers questions about **Generative AI, Large Language Models (LLMs), Transformers, RAG, Prompt Engineering, Fine-Tuning, AI Agents, and more**.

Built with **Python, Tkinter, NLTK, and Scikit-Learn**, it combines an intelligent NLP-powered knowledge engine with a visually stunning **Synthwave-inspired interface**.

---

## 🚀 Features

- 🤖 AI & LLM Knowledge Assistant
- 🎨 Modern Synthwave UI Design
- 🔍 TF-IDF + Cosine Similarity Search
- 🧠 NLP Preprocessing with NLTK
- 📊 Confidence Score Visualization
- ⚡ Real-Time Typing Animation
- 🌆 Animated Neon City Background
- 📚 25+ Curated AI & LLM Topics

## 🎯 How It Works

1. User enters a query.
2. Text is preprocessed using NLTK.
3. TF-IDF vectors are generated.
4. Cosine similarity finds the closest FAQ match.
5. The most relevant answer is displayed with a confidence score.
